## Development

### Prerequisites

- Node.js 18+
- npm or yarn
- Expo CLI

### Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
